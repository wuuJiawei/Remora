Remora supports flexible tab and split pane layouts, allowing you to manage multiple sessions simultaneously.

## Tab Operations

### Basic Operations

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | New tab |
| `Cmd+W` | Close current tab |
| `Cmd+[` | Previous tab |
| `Cmd+]` | Next tab |
| `Cmd+Shift+[` | Move tab left |
| `Cmd+Shift+]` | Move tab right |
| `Cmd+1-9` | Switch to tab number |

### Tab Context Menu

Right-click a tab to:
- Close tab
- Close other tabs
- Close all tabs
- Rename tab
- Duplicate tab content

## Split Pane Operations

Split panes let you view multiple terminals side by side within a single tab.

### Creating Splits

| Shortcut | Action |
|----------|--------|
| `Cmd+D` | Split horizontally |
| `Cmd+Shift+D` | Split vertically |

### Navigating Splits

| Shortcut | Action |
|----------|--------|
| `Cmd+Alt+←` | Focus left pane |
| `Cmd+Alt+→` | Focus right pane |
| `Cmd+Alt+↑` | Focus upper pane |
| `Cmd+Alt+↓` | Focus lower pane |

### Resizing Splits

Drag the split divider to resize panes. Double-click to equalize.

## Layout Combinations

### Common Layouts

- **Multi-tab mode**: One session per tab, great for managing multiple connections
- **Single tab, multiple panes**: Side-by-side terminals in one tab, perfect for monitoring
- **Mixed mode**: Combine tabs and panes for maximum efficiency

### Layout Persistence

Remora automatically saves your workspace layout and restores it on next launch.
