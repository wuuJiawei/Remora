Remora provides multiple ways to quickly connect to your frequently used SSH hosts.

## Quick Connect Panel

Press `Cmd+Shift+K` or `Cmd+K` to open the Quick Connect panel.

### How to Use

1. Open the panel with the keyboard shortcut
2. Enter hostname, IP, or search existing hosts
3. Press Enter to connect to the selected host
4. Favorited hosts display a star icon

### Search Syntax

- Type hostname for fuzzy search
- Enter `user@host` to specify username
- Enter `:port` to specify port

## Favorite Hosts

Add frequently used hosts to favorites for quick access.

### Adding Favorites

1. Right-click a host in the host list
2. Select **Add to Favorites**
3. Or toggle **Favorite** in host details

### Favorite Shortcuts

Favorited hosts appear at the top in Quick Connect. Press number keys `1-9` for quick selection.

## Quick Commands

Quick Commands let you execute frequently used operations with just a few keystrokes.

### Built-in Commands

| Command | Description |
|---------|-------------|
| `/clear` | Clear terminal |
| `/reconnect` | Reconnect current session |
| `/transfer` | Open file transfer panel |

### Custom Quick Commands

Create custom commands in **Settings > Quick Commands**:

1. Click **+** to add a new command
2. Enter command content and optional shortcut
3. Type `/` in terminal to activate Quick Commands panel
