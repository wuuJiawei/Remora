---
title: 快速连接
description: 使用快速连接功能快速访问常用主机。
---

Remora 提供多种快速连接方式，帮助您快速访问常用 SSH 主机。

## 快速连接面板

按 `Cmd+Shift+K` 或 `Cmd+K` 打开快速连接面板。

### 使用方式

1. 按快捷键打开面板
2. 输入主机名、IP 或搜索已有主机
3. 选择目标主机后按 Enter 即可连接
4. 支持收藏的主机会显示星标

### 搜索语法

- 直接输入主机名进行模糊搜索
- 输入 `user@host` 指定用户名
- 输入 `:port` 指定端口

## 收藏主机

将常用主机添加到收藏，方便快速访问。

### 添加收藏

1. 在主机列表中右键点击主机
2. 选择 **添加到收藏**
3. 或在主机详情中勾选 **收藏** 选项

### 收藏快捷键

收藏的主机可以在快速连接面板中优先显示，按数字键 `1-9` 快速选择。

## 快速命令

快速命令让您只需几次按键即可执行常用操作。

### 内置命令

| 命令 | 说明 |
|------|------|
| `/clear` | 清空终端 |
| `/reconnect` | 重新连接当前会话 |
| `/transfer` | 打开文件传输面板 |

### 自定义快速命令

在 **设置 > 快速命令** 中创建自定义命令：

1. 点击 **+** 添加新命令
2. 输入命令内容和可选快捷键
3. 在终端中输入 `/` 激活快速命令面板
