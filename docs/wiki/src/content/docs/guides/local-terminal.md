Remora 不仅支持 SSH 远程连接，还提供完整的本地终端功能。

## 启动本地终端

### 新建本地会话

1. 点击侧边栏的 **+** 按钮
2. 选择 **新建本地终端**
3. 或使用快捷键 `Cmd+N`

### 与 SSH 终端的区别

| 特性 | 本地终端 | SSH 终端 |
|------|----------|----------|
| 运行位置 | 本地 macOS | 远程服务器 |
| 启动命令 | 可自定义 | SSH 连接 |
| 环境变量 | 继承本地环境 | 远程服务器环境 |
| 文件系统 | 本地文件系统 | 远程服务器文件系统 |

## 配置本地终端

在 **设置 > 终端** 中可以配置：

### 启动 Shell

选择默认启动的 Shell：

```bash
# 使用 zsh（默认）
/bin/zsh

# 使用 bash
/bin/bash

# 使用 fish
/usr/local/bin/fish
```

### 环境变量

可以设置自定义环境变量，这些变量会在每次启动终端时加载。

### 提示符自定义

Remora 会尊重您现有的 Shell 提示符配置（如 .zshrc 中的 PS1）。

## 本地终端功能

- **完整终端模拟**：支持 ANSI/VT、256 色、真彩色
- **TUI 兼容**：支持 vim、tmux、htop 等
- **历史记录**：继承 Shell 历史（如 .zsh_history）
- **自动补全**：支持 Shell 内置补全

## 使用场景

- 本地开发测试
- 快速执行本地命令
- 与 SSH 会话配合使用（分栏布局）
- 文件管理（与 SFTP 对比）
