Remora 支持从多种来源导入主机配置，也支持导出数据备份。

## 导入配置

### 从 SSH Config 导入

Remora 可以读取 `~/.ssh/config` 文件中的主机配置：

1. 打开 **设置 > 主机**
2. 点击 **导入**
3. 选择 **从 SSH Config 导入**
4. 选择要导入的主机
5. 点击 **导入**

### 导入格式

Remora 支持以下导入格式：

| 格式 | 说明 |
|------|------|
| SSH Config | `~/.ssh/config` |
| Remora 导出 | `.remora` 文件 |
| JSON | 通用 JSON 格式 |

### JSON 格式示例

```json
{
  "hosts": [
    {
      "name": "My Server",
      "host": "192.168.1.100",
      "port": 22,
      "username": "admin",
      "auth": "password"
    }
  ]
}
```

## 导出数据

### 导出为 Remora 格式

导出所有主机配置为 `.remora` 文件：

1. 打开 **设置 > 主机**
2. 点击 **导出**
3. 选择 **导出为 Remora 格式**
4. 选择保存位置

导出的文件包含：

- 主机配置（不含密码）
- 主机分组
- 快速命令
- 快速路径

### 导出为 JSON

导出为通用 JSON 格式，便于其他工具处理：

1. 打开 **设置 > 主机**
2. 点击 **导出**
3. 选择 **导出为 JSON**

## 数据迁移

### 迁移到新设备

1. 在旧设备上导出配置
2. 传输导出文件到新设备
3. 在新设备上导入配置

### 备份建议

定期备份您的配置，建议：

- 每次大幅修改配置后备份
- 导出文件妥善保管
- 记住 Keychain 密码
