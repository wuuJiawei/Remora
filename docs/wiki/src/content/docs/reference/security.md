---
title: 安全设置
description: 了解 Remora 的安全机制和配置选项。
---

Remora 采用本地优先的安全策略，保护您的凭据和连接安全。

## 凭据存储

### Keychain 存储（推荐）

Remora 使用 macOS Keychain 安全存储密码：

- 密码加密存储在系统 Keychain 中
- 退出应用后凭据不会泄露
- 首次连接时询问是否保存

### 临时存储

也可以选择仅在会话期间存储密码：

- 密码仅保存在内存中
- 关闭应用或断开连接后清除
- 适合公共设备

## SSH 密钥

### 支持的密钥格式

Remora 支持以下 SSH 密钥格式：

- RSA (2048/4096 位)
- ED25519
- ECDSA (256/384/521 位)

### 密钥位置

密钥文件通常存放在 `~/.ssh/` 目录：

```
~/.ssh/id_rsa
~/.ssh/id_ed25519
~/.ssh/id_ecdsa
```

### 密钥权限

确保私钥文件权限正确：

```bash
chmod 600 ~/.ssh/id_ed25519
```

## 主机指纹

### StrictHostKeyChecking

首次连接新主机时，Remora 会显示主机指纹供您确认：

- **询问（默认）**：显示指纹，确认后保存
- **接受**：自动接受并保存指纹
- **拒绝**：拒绝连接

### 管理已知主机

在 **设置 > SSH > 安全** 中可以：

- 查看已保存的主机指纹
- 删除单个或所有已保存的指纹
- 重新验证主机指纹

## 敏感操作确认

Remora 会要求确认以下敏感操作：

- 复制明文密码到剪贴板
- 导出密码
- 关闭密钥存储
- 删除已保存的主机

## 安全日志

在 **设置 > 高级 > 调试日志** 中可以查看安全相关日志，便于排查问题。
