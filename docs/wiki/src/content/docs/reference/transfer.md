---
title: 传输管理
description: 管理 SFTP 文件传输任务。
---

Remora 提供完整的文件传输管理功能。

## 传输队列

所有 SFTP 传输操作都会进入队列统一管理。

### 查看队列

点击侧边栏底部的 **传输** 按钮打开传输面板。

###状态

| 状态 | 说明 |
|------|------|
| 等待中 | 等待传输 |
| 传输中 | 正在传输 |
| 已完成 | 传输成功 |
| 失败 | 传输失败 |

## 传输操作

### 取消传输

点击取消按钮终止传输。

### 重试失败传输

点击重试按钮重新传输失败的文件。

## 并发传输控制

Remora 支持并发传输多个文件，充分利用网络带宽。

### 并发数配置

支持自定义同时传输的文件数量，优化传输效率。

智能队列管理，确保传输公平有序。

## 传输冲突处理

当目标文件已存在时，Remora 提供多种处理策略。

### 冲突策略

| 策略 | 说明 |
|------|------|
| 跳过 | 跳过已存在的文件 |
| 覆盖 | 覆盖已存在的文件 |
| 重命名 | 自动重命名新文件 |

> 敬请期待：更多高级选项

## 传输方向

Remora 支持双向传输：上传和下载。

### 上传

从本地上传到远程服务器。

### 下载

从远程服务器下载到本地。

在 **设置 > 文件管理器** 中可以配置传输相关选项。
