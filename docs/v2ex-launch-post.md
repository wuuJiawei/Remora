# V2EX 宣传稿

## 标题候选

`[自荐] 做了一个 macOS 上原生的 SSH + SFTP 工具 Remora，想听听大家最需要什么功能`

## 正文

最近在做一个 macOS 上的原生 SSH + SFTP 工具，叫 Remora。  
我自己平时长期在终端、服务器、文件传输这些场景里切来切去，越用越觉得现有工具要么不够原生，要么在 TUI 和大滚动场景下体验一般，所以干脆自己做了一个。

我自己遇到的痛点主要有几个：

1. SSH 和 SFTP 经常是割裂的，连上机器后还要再切文件管理工具。
2. 很多终端工具在跑 `vim`、`htop`、`k9s`、`btop` 这类 TUI 时，细节体验差一些。
3. 一旦输出很多、滚动很重，性能和顺滑度就容易掉下来。
4. 有些工具功能很多，但 macOS 上总有一点“不像原生 app”。

所以 Remora 现在想先解决的是：  
把 SSH 会话、SFTP 文件管理、传输流程放到一个原生工作区里，减少来回切换；同时把终端本身的 TUI 兼容性和重滚动性能做扎实。

目前这个工具的一些方向是：

- 原生 UI：SwiftUI + 自己的终端引擎，尽量保留 macOS 应该有的感觉。
- TUI 友好：重点照顾 `vim`、`top/htop`、全屏终端应用这类场景。
- 重滚动性能：大输出、长日志、频繁滚动时，尽量保持顺滑。
- SSH + SFTP 一体化：连机器、看文件、上传下载，尽量在一个地方完成。

后面也会继续做一些更实用的能力，比如：

- 跳板机 / 多级连接
- 导入 `~/.ssh/config`
- 更完整的密钥管理
- 同步/共享连接配置
- AI 支持：这一块已经在开发中，计划先从服务器辅助操作、命令解释、日志理解这类方向切入

想先来 V2EX 听听真实需求。  
如果你平时也会大量用 SSH / SFTP / Terminal，最希望我优先做的 3 个功能是什么？

比如：

- 跳板机
- 导入 ssh config
- 密钥管理
- 会话同步
- 标签页 / 分栏增强
- AI 辅助操作
- 其他你觉得“没它根本不想换工具”的功能

如果有人愿意试用和拍砖，我也很欢迎。直说问题就行，这种反馈最有价值。
