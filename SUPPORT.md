# Support

## Usage Questions

For setup help, workflow questions, and troubleshooting:

- Open a GitHub issue: <https://github.com/wuuJiawei/Remora/issues>
- Or contact the maintainer on X: <https://x.com/1Javeys>

When asking for help, include:

- macOS version
- Remora version or commit
- SSH/SFTP auth mode in use
- reproduction steps
- screenshots or logs that do not expose secrets

## Bug Reports

Use GitHub Issues for reproducible bugs and regressions:

<https://github.com/wuuJiawei/Remora/issues>

Include expected behavior, actual behavior, and any recent changes that may be relevant.

## Security Reports

Do not report vulnerabilities in a public issue.

Send security reports to `wujiawei0926@gmail.com` and follow the process in
`SECURITY.md`.

## Response Expectations

Remora is maintained on a best-effort basis. Response times may vary, but clear
reproduction details usually lead to faster triage.
